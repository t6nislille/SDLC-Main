Git and GitHub

The git commit command captures a snapshot of the project's currently staged changes.

Command

$git commit

Record changes to the repository.

The git init command creates an empty Git repository. init can be used to convert an existing project into a Git repository. The init command can also initialize an empty repository for a new project.

Command

$git init

Create an empty Git repository or reinitialize an existing one

The git remote command lets you create, view, and delete connections to other repositories. Remote connections are more like bookmarks rather than direct links into other repositories.

Command

$git remote add

Manage set of tracked repositories

Let's you create, view and delete connections to other repositories.

Command

$git remote

Manage the set of repositories whose branches you track.

Git branch helps you find out your current branche.

Command

$git branch

List, create or delete branches.

Git clone is used to target an existing repository and create a clone or copy.

Command

$git clone

Clone a repository into a new directory.

The git add command is used to add file contents to the index. It also prepares for the next commit.

Command

$git add

Add file contents to the index.

Helps to go back to the last commit we did and to get rid of all unstaged changes.

Command

$git checkout

Go back to the last commit.

Pushing is how you transfer commits from your local repository to a remote repository.

Command

$git push

Uploads your commits to the remote repository.

The git pull command is used to fetch and download content from a remote repository.

Command

$git pull

Fetch from and integrate with another repository or a local branch.
